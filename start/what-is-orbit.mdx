Orbit gives every AI agent you run a shared brain: one memory to read from, one place to file work, and one live dashboard so you always know what your agents are doing and why. It replaces the copy-paste shuffle between Claude, Codex, n8n, and a dozen browser tabs with a single control layer your whole AI workforce reads before it acts.

## The problem Orbit solves

Right now, every AI tool you use starts from zero. A new Claude chat doesn't know what yesterday's chat figured out. Your n8n workflow doesn't know the decision you made last week. Your Codex run has never heard of the rule you keep repeating.

So you become the glue. You copy context between tools. You re-explain the same constraints. You hope each agent remembers — and it never does.

## What Orbit does instead

Orbit sits in the middle as a shared layer every agent connects to:

- **Agents read before they act.** At the start of every run, an agent pulls your active decisions and approved knowledge.
- **Agents file what they produce.** Every output lands in your Vault — searchable, permanent, visible.
- **You see everything live.** A dashboard shows what each agent did, when, and why.
- **Decisions stick.** File a rule once and every future run respects it. No re-explaining.

## What Orbit is *not*

- **Not another chatbot.** Orbit doesn't replace Claude — it makes Claude (and every other agent) remember and coordinate.
- **Not a prompt manager.** It's a memory and control layer, not a prompt library.
- **Not just for engineers.** If you operate AI tools, Orbit is for you — no code required to get value.

## Who it's for

Solopreneurs running five AI tools at once. Founders building an AI-native team. Operators who live inside Claude all day. If your work is spread across multiple agents and you're tired of being the memory between them, Orbit is built for you.
