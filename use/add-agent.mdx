---
title: "How to add a new agent"
description: "Connect a new AI agent and give it a permanent identity."
---

Adding an agent means connecting an AI tool to Orbit and giving it a role. Once connected, it appears in your workspace with a persistent identity.

## Steps

1. Connect the tool via MCP — see the connection guide for your tool ([Claude](/use/connect-claude), [Claude Code](/use/connect-claude-code), [Codex](/use/connect-codex), [Cursor](/use/connect-cursor), [n8n](/use/connect-n8n)).
2. Run the agent once. On its first `orbit_heartbeat` call, it registers and appears in your **Orbits** list with a **Live** badge.
3. Open the agent and set its identity:
   - **Name** — how it shows on your dashboard
   - **Role title** — e.g. "Research Analyst"
   - **Department** — Sales, Research, Content, Ops, or custom
   - **Project** — the context bucket it files into
4. Save. The agent now appears in your [org chart](/use/organize-org-chart).

## Give it a stable identity

An agent should use the same `agent.id` on every run so Orbit tracks it as one continuous identity rather than creating duplicates. Agents using the [Orbit MCP Skill](/use/mcp-skill) handle this automatically.
