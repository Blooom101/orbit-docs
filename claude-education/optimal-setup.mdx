You've learned the surfaces, models, folders, permissions, commands, and schedules. This page ties them into one setup where every piece of Claude work makes the next piece smarter — because Orbit is the shared brain underneath all of it.

## The optimal setup, in order

1. **Connect Orbit everywhere.** Claude chat, Claude Code (local and cloud), and every Cowork agent point at the same Orbit workspace. One Vault, one set of decisions. ([Connect Claude](/use/connect-claude), [Connect Claude Code](/use/connect-claude-code))
2. **Install the Orbit MCP Skill** on every agent, so each one heartbeats, files results at the right trust level, and searches before working — automatically. ([MCP Skill](/use/mcp-skill))
3. **Set "Always allow"** for Orbit tools so the heartbeat loop never breaks. ([Permissions](/claude-education/permissions))
4. **File your standing rules as decisions**, not buried in one repo's `CLAUDE.md`. Company-wide policy → [Orbit decisions](/concepts/decisions); repo mechanics → `CLAUDE.md`.
5. **Right-size models per task and cadence.** Sonnet as daily driver, Haiku for frequent background jobs, Opus for the hard stuff. ([Models](/claude-education/models))
6. **Put recurring work on schedules** via Cowork, each reporting into Orbit. ([Routines](/claude-education/routines-schedules))
7. **Live in your Briefing and Inbox.** Review what every surface produced, approve what becomes ground truth. ([Briefing](/use/read-briefing), [Inbox](/use/process-inbox))

## What this gets you

- **No more starting from zero.** Every chat, Code session, and scheduled run reads your decisions and prior conclusions first.
- **No more re-explaining.** Decide once; every surface honors it forever.
- **No more black boxes.** Unattended agents are as visible as your live sessions.
- **Compounding output.** Today's work is tomorrow's context. Quality climbs over time instead of resetting each session.

## A day in the optimal setup

- **Morning:** open your Briefing. Overnight Cowork agents filed digests and drafts. Clear your Inbox, approving what's good into the Vault.
- **Midday:** drive a Claude Code session on a feature. It reads your architecture decisions, branches, opens a PR, files a summary.
- **Afternoon:** a chat session researches a question — and pulls in what this morning's agents already found via Vault search.
- **Anytime:** you change a decision once; every surface picks it up on its next run.

## The mental model, one more time

> **Claude reads context from Orbit → does the work → files results back → you review → it becomes context next time.**

Run that loop across chat, Code, and Cowork, every day, and you're not just using Claude — you're operating an AI workforce that gets smarter with every run.
