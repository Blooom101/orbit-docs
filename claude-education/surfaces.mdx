Claude shows up in three main surfaces. Knowing which to use for what is the single biggest lever on how productive you'll be. All three connect to the same Orbit Vault.

## 1. Claude chat (claude.ai + Projects)

The conversational surface. You type, Claude responds. **Projects** add a persistent workspace with shared files and instructions.

**Reach for it when:** thinking, writing, strategy, research, drafting, one-off questions.

**With Orbit:** connect Orbit as an MCP server and every chat reads your decisions and files its outputs to the Vault. See [Connect Claude](/use/connect-claude).

## 2. Claude Code (the agentic surface)

Claude with hands. It reads and edits files, runs commands, uses tools, and works across a whole project — in your terminal, the desktop app, the web (claude.ai/code), or your IDE.

**Reach for it when:** building software, editing real files, multi-step tasks that touch your filesystem or repo.

**With Orbit:** Claude Code reads your architectural decisions before each session and files code outputs and summaries to the Vault. See [Connect Claude Code](/use/connect-claude-code) and [Claude Code basics](/claude-education/claude-code-basics).

## 3. Cowork (scheduled agents)

Claude that runs on a **timer**, without you starting it. A Cowork agent wakes on its cadence, does its job, and goes back to sleep.

**Reach for it when:** recurring work — a morning digest, daily outreach, a weekly report.

**With Orbit:** the agent heartbeats at the start and end of every run, files results to your Vault, and shows up in your [Briefing](/use/read-briefing). See [Cowork](/claude-education/cowork).

## Quick chooser

| You want to… | Use |
|---|---|
| Think, write, research | **Chat** |
| Build or edit a real project | **Claude Code** |
| Run something on a schedule | **Cowork** |

## They all share one brain

The point of connecting Orbit is that these three surfaces stop being separate. A decision you file in chat is read by Claude Code. Work your Cowork agent produces overnight is searchable by your next chat. One Vault underneath all three.
