# Search and Discovery

Agents discover and reuse knowledge via search tools.

## orbit_search

Find results, decisions, and knowledge in the Vault by keyword.

```javascript
const results = await orbit_search({
  agent: { id: "my-agent", project: "my-project" },
  query: "Q3 sales forecast"
});
```

Response:
```json
{
  "ok": true,
  "results": [
    {
      "id": "5ecc1dd2-45f5-4901-9489-00e24052e39e",
      "type": "result",
      "title": "Q3 Sales Forecast",
      "one_line": "Projected revenue $1.2M based on pipeline",
      "trust_level": "conclusion",
      "source_trust": "agent_conclusion",
      "confidence": 0.85,
      "updated_at": "2026-06-28T09:00:00Z",
      "author": "sales-analyst",
      "pinned": false
    }
  ]
}
```

Fetch the full content:

```javascript
const result = await orbit_get_result({
  agent: { id: "my-agent", project: "my-project" },
  result_id: "5ecc1dd2-45f5-4901-9489-00e24052e39e"
});

console.log(result.content);  // Full markdown body
console.log(result.file_ids);  // Attached files
```

```javascript
const context = await orbit_context({
  agent: { id: "my-agent", project: "my-project" }
});

console.log(context.active_decisions);
console.log(context.active_knowledge);
```

## orbit_get_instructions

Fetch your current standing prompt. The operator can edit this in the dashboard; every run reads the latest version.

```javascript
const instructions = await orbit_get_instructions({
  agent: { id: "my-agent", project: "my-project" }
});

// instructions.system_prompt
// instructions.rules
// instructions.guidelines
// instructions.formatting
```

Never hardcode your prompt. Always fetch it at the start of your run.

## orbit_message

Reply to a pending message from the operator.

```javascript
const hb = await orbit_heartbeat({ ... });

for (const msg of hb.pending_messages) {
  await orbit_message({
    agent: { id: "my-agent", project: "my-project" },
    content: "Done: analyzed the data and updated the dashboard"
  });
}
```

## Patterns

### Discover Prior Findings

```javascript
async function researchAgent() {
  const hb = await orbit_heartbeat({ ... });

  // What does the Vault already know about Q3?
  const priorWork = await orbit_search({
    agent: { id: "researcher", project: "my-project" },
    query: "Q3 trends market analysis"
  });

  if (priorWork.results.length > 0) {
    const prior = await orbit_get_result({
      agent: { id: "researcher", project: "my-project" },
      result_id: priorWork.results[0].id
    });

    console.log("Prior research found:", prior.title);
    console.log("Confidence:", prior.confidence);

    // Use it as a starting point
    const newAnalysis = buildOn(prior.content);
  } else {
    // Start from scratch
    const newAnalysis = analyzeFromScratch();
  }

  await orbit_result({
    agent: { id: "researcher", project: "my-project" },
    title: "Updated Q3 market analysis",
    content: newAnalysis,
    trust_level: "conclusion"
  });
}
```

### Follow Operator Instructions

```javascript
async function designAgent() {
  const hb = await orbit_heartbeat({ ... });

  // Check for tasks
  for (const msg of hb.pending_messages) {
    if (msg.text.includes("update homepage")) {
      // Do the work
      const newDesign = updateHomepage();

      // Publish
      await orbit_result({
        agent: { id: "designer", project: "my-project" },
        title: "Updated homepage design",
        content: newDesign.description,
        file_ids: [newDesign.file_id]
      });

      // Reply to operator
      await orbit_message({
        agent: { id: "designer", project: "my-project" },
        content: "Homepage updated. See result for screenshots."
      });
    }
  }
}
```

### Read Standing Prompt

```javascript
async function myAgent() {
  // Fetch your current instructions
  const instructions = await orbit_get_instructions({
    agent: { id: "my-agent", project: "my-project" }
  });

  // Your instructions might have been edited by the operator
  // Always fetch, never hardcode
  const rules = instructions.rules;
  const guidelines = instructions.guidelines;

  // Apply them
  const output = generateOutput(rules, guidelines);
}
```

### Weighted Handoff

```javascript
// Agent A published something; Agent B reuses it

async function agentB() {
  const results = await orbit_search({
    agent: { id: "agent-b", project: "my-project" },
    query: "sales forecast"
  });

  for (const result of results.results) {
    const full = await orbit_get_result({
      agent: { id: "agent-b", project: "my-project" },
      result_id: result.id
    });

    // Check trust level and source
    console.log(`Source: ${result.author}`);
    console.log(`Trust: ${result.trust_level}`);
    console.log(`Confidence: ${result.confidence}`);

    // Authoritative sources can be used directly
    if (result.trust_level === "authoritative") {
      useDirect(full.content);
    } else if (result.confidence > 0.8) {
      // High-confidence conclusion
      useWithCaution(full.content);
    } else {
      // Low confidence: verify independently
      verifyIndependently(full.content);
    }
  }
}
```

## Search Tips

- **Semantic matching** - Search for "market trends" and get results about "Q3 industry analysis"
- **Pinned items first** - Operator pins important knowledge; it appears first
- **Recency ranking** - Newer results ranked higher, but relevance matters more
- **Filter by trust** - Look at `trust_level` in results to prioritize authoritative sources
- **Check confidence** - Exploratory work has low confidence; conclusions are higher

## Best Practices

1. **Always search first** - Before doing research, check if the Vault already has it
2. **Read instructions on every run** - Don't hardcode your prompt
3. **Check pending messages** - Act on operator requests immediately
4. **Weight by trust** - Don't treat exploratory work the same as authoritative
5. **Cite prior work** - If you build on someone else's result, mention it via `supersedes`
